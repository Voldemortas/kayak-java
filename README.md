# Task1

Prints frequency graph

Run `Task1`

## Example 1

Input: `1, 4, 1, 5, 8, 1, 3, 5, 1, 4, 1, 3, 7, 2`

Output:
```
*       
*       
*       
* ***   
***** **
12345678
```

## Example 2

Input: `9, 7, 7, 11, 11`

Output:
```
*   *
* * *
   11
78901
```

-----

## Footnote

Numbers are printed in vertical manner - 
```
1222
9012
```
shows `19, 20, 21, 22`
